# TicTacToe

A complete Tic Tac Toe game in Godot with a focus on clean architecture and user interaction. Implemented core gameplay systems including turn management, win/draw detection, strike-line visualization, menu navigation, and audio feedback, while maintaining a clear separation between logic and UI for scalability.

### Game Design Document: [Click Here](https://github.com/ankit-suman-07/GODOT_TicTacToe/blob/main/GDD.md)
### Credits: [Click Here](https://github.com/ankit-suman-07/GODOT_TicTacToe/blob/main/Credits.md)

## 💻 Setup locally

**Prerequisites** :
- Git 
- Godot v4.6.1

**Step 1:** Clone the repository
```
git clone https://github.com/ankit-suman-07/GODOT_TicTacToe.git
```

**Step 2:** Open Godot
```
In godot editor, click open a new project and browse to the cloned repository folder and select.
```
